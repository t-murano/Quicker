# Simple chat application with Twitter login

Realtime chat application with Firebase 

## Used modules
- [jQuery](https://jquery.com) 2.2.3
- [Firebase](www.firebase.com) 2.4.2
- [Gulp](gulpjs.com) 3.9.1
- [Moment.js](momentjs.com) 2.13.0

## Getting started

```bash
npm install && gulp
```
